Virtual DAA Lab is a web-based frontend project designed to simulate an interactive learning platform for Design and Analysis of Algorithms.
The application allows users to log in, navigate through a dashboard, watch algorithm-related educational videos, 
and track their learning progress dynamically. It is built using HTML, CSS, and JavaScript, with localStorage used to 
manage user data and progress without a backend. The project follows a modular structure with separate pages for login, home, videos, and progress,
all connected through a responsive navigation system. Its purpose is to demonstrate core web development concepts such as UI design, DOM manipulation,
state management, and multi-page navigation while providing a foundation for future backend integration using technologies like Spring Boot and MySQL.
